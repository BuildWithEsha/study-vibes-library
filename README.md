# Study Vibes Library 🌼

A cozy, aesthetic resource hub designed for students and learners to access curated study tools, lo-fi playlists, productivity apps, and more — all in one beautiful place.

---

## 🌟 Features

- 📂 Categories: Lo-Fi, Pomodoro, Study Techniques, Tools, Sounds, Favorites
- ➕ Add your own resources or categories
- 📌 Save favorites for quick access
- 💾 LocalStorage for saving without a backend
- 🧠 Fully built with HTML, CSS, and Vanilla JS

---

## 🛠️ Tech Stack

- HTML5 + CSS3
- JavaScript (Vanilla)
- No frameworks, no libraries — just pure frontend ✨
